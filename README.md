# cisco-lab_0.1

Лабораторная работа в среде Cisco Packet Tracer, демонстрирующая настройку ключевых технологий сетевого администрирования: ACL, NAT, OSPF и VLAN.

## 🚀 Описание проекта
Данная лабораторная работа представляет собой комплексную сетевую топологию, в которой реализованы следующие задачи:
* **VLAN (Virtual Local Area Network):** Сегментация сети на логические уровни для повышения безопасности и управления трафиком.
* **OSPF (Open Shortest Path First):** Настройка динамической маршрутизации для обеспечения связности между узлами.
* **NAT (Network Address Translation):** Настройка трансляции адресов для выхода внутренней сети в интернет (или внешнюю сеть).
* **ACL (Access Control Lists):** Применение списков контроля доступа для фильтрации трафика и обеспечения безопасности.

## 🛠 Технологии и инструменты
* **Cisco Packet Tracer** (рекомендуемая версия 8.0 и выше)
* Типы устройств: Router, Switch, End Devices (PC/Server).
